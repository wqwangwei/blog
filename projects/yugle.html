      # 运行步骤

      1. 搭建 Golang 和 node.js 运行环境

      2. git clone https://github.com/YupaiTS/yugle.git 下载代码

      3. 使用 go get xxx 命令下载依赖包

      4. 在本地 MySQL 数据库中新建 yugle 数据库

      5. 使用 go test yugle_test.go 命令运行单元测试用例

      6. 使用 go run main.go 命令运行项目
